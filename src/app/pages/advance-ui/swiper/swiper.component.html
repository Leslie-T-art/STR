Default Swiper

Use ngx-slick-carousel element to set a default swiper.

Navigation Swiper

set arrows = true option to set a swiper with navigation.

Pagination Dynamic Swiper

Set dots: true, option to set a dynamic swiper with pagination.

Vertical Swiper

Set vertical : true option to set a vertical swiper.

Mousewheel Control Swiper

Use mousewheel event to set a swiper with mousewheel scroll.

Effect Fade Swiper

Use fade = true option to set a swiper with fade effect.

Effect Coverflow Swiper

Use centerMode:true centerPadding: '60px' options to set a swiper with coverflow effect.

Responsive Breakpoints Swiper

Use slidesToShow options to set a responsive swiper.
